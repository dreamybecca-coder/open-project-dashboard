## 筛选逻辑修正记录 (2026-04-06)

### 第一次修正
**问题**：筛选出的人很多不符合条件
- 赵海森(19岁)：年轻人✓，但无海外背景✗
- 陈怡(30岁)：年轻人✓，但无海外背景✗  
- 李颖(28岁)：年轻人✓，但无海外背景✗
- 王一(32岁)：年轻人✓，但无海外背景✗

**修正**：
1. 年龄提取：排除过去时态描述（如"16岁赴英国留学"）
2. 在读状态：必须明确"在读"而非历史描述
3. 海外背景：必须有留学国家/学校 + 留学描述
4. 排除工作经验>8年的人

### 第二次修正（重大）
**问题**：李姬韧（花辣子米）有真实海外背景（南加州大学）但完全没被筛选出来

**根本原因**：
1. **只看关键词，不看学校名称** - 之前只搜索"留学"、"海外"等关键词，漏掉了很多直接提到海外学校名称的人
2. **年龄判断过于僵化** - 用32岁作为硬性筛选条件，导致有真实海外背景的人被排除
3. **未正确理解"退学回国"** - "从美国退学回国"恰恰证明了真实的海外留学经历

**修正**：
1. 海外背景检测：增加海外学校名称识别（南加州大学、莱斯大学、斯坦福、哈佛、剑桥等）
2. 年龄推断：支持中文数字（"十年前回国"）和更多推断模式
3. 筛选范围：从"32岁以下+海归"改为"所有海归背景"（不限年龄）
4. 结果：从2人扩展到45人

**关键教训**：
- 不能只依赖关键词匹配，要识别具体的学校名称
- 退学回国也是真实的海外经历，不应排除
- 年龄限制可能过于严格，应优先筛选海外背景
